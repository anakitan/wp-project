## Technologies 
- Java  
- Spring Boot  
- Thymeleaf  
- PostgreSQL  
- HTML/CSS  
- Bootstrap  

## Functionalities of the roles

#### Users with ROLE_USER
- Log in
- View all products
- Add product to shopping cart

#### Users with ROLE_ADMIN
- Log in
- View all products
- Add new product
- Delete the product
- Edit the product
- Add product to shopping cart
- Can add new category
